## 4. フィールド要素一覧

| 番号 |      論理名      |      タイプ      | 活性／表示 |                                 内容                                 |
| ---- | ---------------- | ---------------- | ---------- | -------------------------------------------------------------------- |
| 1    | 行選択ボタン     | ラジオボタン     | 活性       | 登録済SNS一覧テーブル一覧から選択対象を指定すること。                                     |
| 1    | 名称             | ラベル           | -          | 登録済のSNS名称を表示すること。                                      |
| 1    | 状態             | セレクトボックス | 非活性     | 登録済のSNSの活動状況を表示すること。                                |
| 1    | サービスURL      | ラベル           | -          | 登録済のサービスのURLを表示すること。                                |
| 1    | テストアカウント | ラベル           | -          | 登録済のサービスへのアクセスをテストできるアカウントを表示すること。 |
