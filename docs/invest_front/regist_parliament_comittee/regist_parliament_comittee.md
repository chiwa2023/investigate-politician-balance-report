### 6.1 委員会区分

委員会区分は地方自治法：「第百九条　普通地方公共団体の議会は、条例で、常任委員会、議会運営委員会及び特別委員会を置くことができる。」を参照している

|     論理名     |  値  |                 説明                 |
| -------------- | ---: | ------------------------------------ |
| 常任委員会     |    1 | 法律で規定される常任委員会を表す     |
| 議院運営委員会 |    2 | 法律で規定される議院運営委員会を表す |
| 特別委員会     |    3 | 法律で規定される特別委員会を表す     |
